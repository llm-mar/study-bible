**notes:**

- This clause comes from a genealogical sequence (cf. Matthew 1:12). The Greek uses μετοικεσία (“relocation/deportation/exile”) with Βαβυλῶνος (“of/to Babylon”), so the sense is “after Jeconiah/Jechoniah was carried off to Babylon.” Proper names: Ἰεχονίας = Jeconiah/Jechoniah; Σαλαθιήλ = Salathiel; Ζοροβαβέλ = Zorobabel.
