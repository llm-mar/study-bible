**notes:**

- This is a genealogical line (Matthew 1:3). The Greek δὲ (“then/but/and”) is usually smoothed into “and” in English genealogy; it doesn’t imply a strong contrast here. Proper names are transliterated (Ἰούδας = Judah, Φαρὲς = Perez, Ζαρὰ = Zerah, Θάμαρ = Tamar, Ἑσρώμ = Hezron, Ἀράμ = Ram).
